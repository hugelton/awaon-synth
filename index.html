起動できない場合の対処法

初回起動時に「開発元が不明なため開けません」というエラーが表示される場合があります。以下の手順を実行してください：

  1. ターミナルを開く (アプリケーション → ユーティリティ → ターミナル)
  2. 以下のコマンドを実行
    xattr -dr com.apple.quarantine ~/Downloads/Awaon.app

    ※ Awaon.app のパスは実際の保存場所に置き換えてください

  3. Awaon.app を起動

補足: このコマンドは macOS のセキュリティ属性 (quarantine attribute) を削除するもので、2025年でも引き続き有効です。署名されていないアプリケーションやインターネットから直接ダウンロードされたアプリケーションに対して使用されます。

Troubleshooting

If you see an error message "cannot be opened because the developer cannot be verified" on first launch, follow these steps:

  1. Open Terminal (Applications → Utilities → Terminal)
  2. Run the following command
    xattr -dr com.apple.quarantine ~/Downloads/Awaon.app

    Replace the path with the actual location of Awaon.app if different

  3. Launch Awaon.app

Note: This command removes the macOS quarantine attribute and remains effective in 2025. It is used for unsigned applications or those downloaded directly from the internet.

主な機能

オシレーター: ウェーブテーブル合成による 2 つのオシレーター + トランジェントシェイパー

フィルター: 4 種類の VCF (ローパス、ハイパス、バンドパス、ノッチ) と共鳴フィルター

エフェクト: ディレイ、コーラス、リバーブ

モジュレーション: 2 つのエンベロープと 2 つの LFO、完全なモジュレーションマトリックス

プリセット: プリセット管理システムと仮想 MIDI キーボード

Features

Oscillators: 2 wavetable oscillators + transient shaper with wavetable synthesis

Filters: 4 types of VCF (lowpass, highpass, bandpass, notch) and resonance filter

Effects: Delay, Chorus, Reverb

Modulation: 2 envelopes and 2 LFOs with complete modulation matrix

Presets: Preset management system and virtual MIDI keyboard

初回起動時の設定

Awaon Synth は初回起動時に、ユーザープリセットとウェーブテーブル用のディレクトリを自動生成します:

~/Library/Audio/Presets/Hugelton/Awaon/

このディレクトリ内に以下が作成されます:

  • Init.awaon - デフォルトプリセットファイル
  • Wavetables/ - ユーザーウェーブテーブル用フォルダ

カスタムウェーブテーブルは .wtb 拡張子で Wavetables/ フォルダに保存できます。

Initial Setup

On first launch, Awaon Synth automatically creates a directory for user presets and wavetables:

~/Library/Audio/Presets/Hugelton/Awaon/

The following are created in this directory:

  • Init.awaon - Default preset file
  • Wavetables/ - Folder for user wavetables

Custom wavetables can be saved with the .wtb extension in the Wavetables/ folder.
