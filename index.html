Init Preset をインストール（macOS のみ） Install Init Preset (macOS only)

初期プリセットをインストールする手順（macOS のみ）

  1. 以下のファイルをダウンロード：
    • Init.awaon
    • install-preset.scpt
  2. install-preset.scpt をダブルクリック
  3. ファイル選択ダイアログで、ダウンロードした Init.awaon を選択
  4. 自動的に ~/Library/Audio/Presets/Hugelton/Awaon/ に配置されます

注意: 同時に Wavetables/ フォルダも自動作成されます。

Steps to install the initial preset (macOS only)

  1. Download the following files:
    • Init.awaon
    • install-preset.scpt
  2. Double-click install-preset.scpt
  3. In the file dialog, select the downloaded Init.awaon
  4. It will automatically be placed in ~/Library/Audio/Presets/Hugelton/Awaon/

Note: The Wavetables/ folder will also be created automatically.

起動に問題がある場合 Troubleshooting

初回起動時に「開発元が不明なため開けません」というエラーが表示される場合、以下のコマンドを実行してください：

  1. ターミナルを開く (アプリケーション → ユーティリティ → ターミナル)
  2. 以下のコマンドを実行
    xattr -dr com.apple.quarantine ~/Downloads/Awaon.app

    ※ パスは実際の保存場所に置き換えてください

If you see "cannot be opened because the developer cannot be verified", run this command in Terminal:

  1. Open Terminal (Applications → Utilities → Terminal)
  2. Run the command
    xattr -dr com.apple.quarantine ~/Downloads/Awaon.app

    Replace the path if Awaon.app is in a different location
